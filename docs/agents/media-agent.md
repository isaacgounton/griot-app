# Media Agent - Interactive AI-Powered Content Creation

## Overview

The **Media Agent** is an interactive AI agent available in the agents page that guides users through creating professional short-form videos with AI-generated content. It's designed for conversational interaction where users can ask what it can do, request specific tasks, and get real-time guidance through the entire video creation workflow.

## Features

### Core Capabilities

1. **Script Generation** 📝
   - Create engaging scripts from topics or ideas
   - Support for multiple styles: motivational, educational, funny, storytelling, tutorials, etc.
   - Multi-language support (30+ languages)
   - Optimize for text-to-speech and viewer retention
   - Adjust tone, pace, and target audience

2. **Audio Generation (TTS)** 🎤
   - Convert scripts to natural-sounding narration
   - Multiple voices and accents available
   - Support for multiple languages
   - Adjustable speech speed and tone
   - Multiple providers: Kokoro, Edge TTS, and more

3. **Image Generation** 🎨
   - AI-generated custom visuals
   - Multiple art styles: animated, cinematic, realistic, artistic, minimalist
   - Image prompts auto-generated from script content
   - Multiple models: Kontext, FLUX, and others
   - Customizable resolution and dimensions

4. **Video Creation** 🎬
   - Combine images into videos with dynamic effects:
     - **Zoom**: Cinematic zoom effect for emphasis
     - **Pan**: Smooth panning across images
     - **Fade**: Smooth transitions between clips
   - Customizable frame rates and resolution
   - Support for various video lengths (15s to 60s+)
   - Automatic timing sync with audio

5. **Audio Mixing** 🎵
   - Mix TTS narration with video
   - Add background music from library
   - Balance audio levels (narration vs music)
   - Adjust music volume independently
   - Match audio duration to video length

6. **Captioning & Styling** 📹
   - Add styled captions to videos
   - Customize: font, size, color, position, style
   - Highlight important keywords
   - Support for modern, bold, classic, and minimalist styles
   - Multi-language subtitle support

7. **Social Media Posting** 📱
   - Post to multiple platforms via Postiz integration:
     - TikTok
     - Instagram Reels
     - YouTube Shorts
     - Facebook Reels
     - LinkedIn Video
   - Auto-generate platform-specific captions and hashtags
   - Schedule posts for optimal engagement
   - Track performance metrics

### Full Video Generation Workflow

When creating a complete video, the agent will:

#### **Stage 1: Planning** 📋

- Clarify the topic/concept
- Confirm language and voice preferences
- Determine visual style
- Set video duration and platform

#### **Stage 2: Content Creation** ✍️

- Generate engaging script
- Ask for approval/refinements
- Generate TTS audio
- Create image prompts from script

#### **Stage 3: Visual Generation** 🎨

- Generate AI images for each scene
- Show you the images
- Offer to refine if needed

#### **Stage 4: Video Production** 🎬

- Create video clips from images with effects
- Add captions to each clip
- Mix audio with videos
- Merge all clips into single video

#### **Stage 5: Finalization** 🎵

- Add background music
- Balance audio levels
- Generate social media captions

#### **Stage 6: Distribution** 📱

- Post to selected platforms via Postiz
- Generate platform-specific hashtags
- Provide final video URLs

## Troubleshooting

### Issue: Video generation is slow

- **Solution**: Check file sizes of generated images
- **Tip**: Shorter videos (15-30s) generate faster

### Issue: Audio doesn't match video length

- **Solution**: Request audio remix from the agent
- **Say**: "The audio is too long/short for the video"

### Issue: Images don't match the script

- **Solution**: Request image regeneration
- **Say**: "Regenerate images with more [style] aesthetic"

### Issue: Captions are hard to read

- **Solution**: Ask for caption style change
- **Say**: "Change captions to bold white with black outline"
